# DSPYurikoPlugin

自用的戴森球计划(Dyson Sphere Program)插件

## Roadmap
- [x] 垃圾箱。储物箱上限调成0，此时可以销毁进入储物箱的物品
- [x] AssembleComponent(熔炉x2/工作台x2/化工厂x10/原油精炼厂x10/粒子对撞机x10) 加速生产
- [x] 传送带加速，约 100/s
- [x] 分拣器加速x10
- [x] 分馏塔, 概率x2
- [x] 实验室制作x2
- [x] 轨道弹射器x2
- [x] 垂直发射井x2
- [x] 物流站飞机速度x2
- [x] 轨道采集器x10
- [x] 射线接收器。暖机x2，光子x2
- [x] 星球矿机。在运输站中设置了行星上有的矿物，就能以 3600/min * 挖矿效率 的速度挖取矿物。
